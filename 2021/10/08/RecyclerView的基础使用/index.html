RecyclerViewзҡ„еҹәзЎҖдҪҝз”Ё

еҸ‘иЎЁдәҺ2021-10-08|жӣҙж–°дәҺ2021-10-08|AndroidеҺҹз”ҹ
|еӯ—ж•°жҖ»и®Ў:2k|йҳ…иҜ»ж—¶й•ҝ:7еҲҶй’ҹ|йҳ…иҜ»йҮҸ:

дёҖгҖҒRecyclerViewжҰӮи§Ҳ

жқҘиҮӘAndroidејҖеҸ‘иҖ…ж–ҮжЎЈпјҡ

RecyclerView еҸҜд»Ҙи®©жӮЁиҪ»жқҫй«ҳж•Ҳең°жҳҫзӨәеӨ§йҮҸж•°жҚ®гҖӮжӮЁжҸҗдҫӣж•°жҚ®е№¶е®ҡд№үжҜҸдёӘеҲ—иЎЁйЎ№зҡ„еӨ–и§ӮпјҢиҖҢ RecyclerView еә“дјҡж №жҚ®йңҖиҰҒеҠЁжҖҒеҲӣе»әе…ғзҙ гҖӮ

йЎҫеҗҚжҖқд№үпјҢRecyclerView дјҡеӣһж”¶иҝҷдәӣеҚ•дёӘзҡ„е…ғзҙ гҖӮеҪ“еҲ—иЎЁйЎ№ж»ҡеҠЁеҮәеұҸе№•ж—¶пјҢRecyclerView дёҚдјҡй”ҖжҜҒе…¶и§ҶеӣҫгҖӮзӣёеҸҚпјҢRecyclerView дјҡеҜ№еұҸе№•дёҠж»ҡеҠЁзҡ„ж–°еҲ—иЎЁйЎ№йҮҚз”ЁиҜҘи§ҶеӣҫгҖӮиҝҷз§ҚйҮҚз”ЁеҸҜд»Ҙжҳҫи‘—жҸҗй«ҳжҖ§иғҪпјҢж”№е–„еә”з”Ёе“Қеә”иғҪеҠӣе№¶йҷҚдҪҺеҠҹиҖ—гҖӮ

RecyclerViewиў«зңӢдҪңжҳҜListViewе’ҢGridViewзҡ„з»§жүҝиҖ…пјҢе…¶дёӯдёҖдёӘеҺҹеӣ жҳҜе®ғе…·жңүжӣҙеҘҪзҡ„еҸҜжӢ“еұ•жЎҶжһ¶пјҢзү№еҲ«жҳҜеӣ дёәе®ғжҸҗдҫӣдәҶе®һзҺ°ж°ҙе№іе’ҢеһӮзӣҙеёғеұҖзҡ„иғҪеҠӣпјҲListViewж— жі•е®һзҺ°ж°ҙе№іж»ҡеҠЁпјүгҖӮеҪ“жҲ‘д»¬зҡ„ж•°жҚ®йӣҶеҗҲйңҖиҰҒеңЁиҝҗиЎҢж—¶еӣ з”ЁжҲ·иЎҢдёәжҲ–зҪ‘з»ңдәӢд»¶иҖҢеҸ‘з”ҹж”№еҸҳж—¶пјҢжҲ‘д»¬йңҖиҰҒдҪҝз”ЁRecyclerViewгҖӮ

еҰӮжһңжғідҪҝз”ЁRecyclerViewпјҢжҲ‘д»¬йңҖиҰҒд»ҘдёӢе·Ҙе…·пјҡ

  • RecycerView.AdapterвҖ”вҖ”еӨ„зҗҶж•°жҚ®йӣҶеҗҲе№¶е°Ҷе…¶з»‘е®ҡеҲ°и§Ҷеӣҫ
  • LayoutManagerвҖ”вҖ”её®еҠ©е®ҡдҪҚз»„д»¶

дёҺListViewзӣёжҜ”пјҢRecyclerViewе…·жңүзҡ„зү№зӮ№пјҡ

  • AdapterдёӯйңҖиҰҒViewHolderпјҡListViewйҖӮй…ҚеҷЁдёҚйңҖиҰҒдҪҝз”Ё ViewHolder жЁЎејҸжқҘжҸҗй«ҳжҖ§иғҪгҖӮзӣёжҜ”д№ӢдёӢпјҢRecyclerViewе®һзҺ°йҖӮй…ҚеҷЁйңҖиҰҒдҪҝз”Ё RecyclerView.ViewholderгҖӮ
  • еҸҜиҮӘе®ҡд№үзҡ„йЎ№зӣ®еёғеұҖпјҡListViewеҸӘиғҪд»ҘеһӮзӣҙзәҝжҖ§жҺ’еҲ—ж–№ејҸеёғеұҖйЎ№зӣ®пјҢе№¶дё”ж— жі•иҮӘе®ҡд№үгҖӮзӣёжҜ”д№ӢдёӢпјҢRecyclerViewжңүдёҖдёӘRecyclerView.LayoutManagerжқҘе…Ғи®ёд»»дҪ•йЎ№зӣ®еёғеұҖпјҢеҢ…жӢ¬ж°ҙе№іеҲ—иЎЁжҲ–дәӨй”ҷзҪ‘ж јгҖӮ
  • йЎ№зӣ®иЈ…йҘ°пјҡListViewе…·жңүandroid:dividerпјҢеҸҜд»ҘеңЁеҲ—иЎЁдёӯзҡ„йЎ№зӣ®д№Ӣй—ҙиҪ»жқҫеҲҶйҡ”зҡ„еұһжҖ§гҖӮиҖҢRecyclerViewйңҖиҰҒдҪҝз”ЁдёҖдёӘRecyclerView.ItemDecorationеҜ№иұЎжқҘи®ҫзҪ®жӣҙеӨҡзҡ„жүӢеҠЁеҲҶйҡ”иЈ…йҘ°гҖӮ

дәҢгҖҒRecyclerViewзҡ„з»„жҲҗ

1гҖҒLayoutManagers

дёҖдёӘRecyclerViewйңҖиҰҒз»‘е®ҡдёҖдёӘеёғеұҖз®ЎзҗҶеҷЁ(LayourManager)е’ҢдёҖдёӘе®һдҫӢеҢ–зҡ„йҖӮй…ҚеҷЁ(Adapter)гҖӮеёғеұҖз®ЎзҗҶеҷЁеҸҜд»Ҙе®ҡдҪҚRecyclerViewдёӯеҗ„дёӘе…ғзҙ зҡ„и§ҶеӣҫгҖӮ

RecyclerViewжҸҗдҫӣиҝҷдәӣеҶ…зҪ®зҡ„еёғеұҖз®ЎзҗҶеҷЁпјҡ

  • LinearLayoutManagerпјҡеңЁеһӮзӣҙжҲ–ж°ҙе№іж»ҡеҠЁеҲ—иЎЁдёӯжҳҫзӨәйЎ№зӣ®гҖӮ
  • GridLayoutManagerпјҡеңЁзҪ‘ж јдёӯжҳҫзӨәйЎ№зӣ®гҖӮ
  • StaggerGridLayoutManagerпјҡд»ҘдәӨй”ҷзҪ‘ж јжҳҫзӨәйЎ№зӣ®гҖӮ

еҰӮжһңжІЎжңүжҳҫжҖ§жҢҮжҳҺLayoutManagerпјҢRecyclerViewе°ҶдёҚдјҡжҳҫзӨәе№¶жҠҘй”ҷE/RecyclerView: No layout manager attached; skipping layoutгҖӮ

2гҖҒRecyclerView.Adapter

RecyclerViewйңҖиҰҒдёҖдёӘж–°зҡ„йҖӮй…ҚеҷЁпјҢе®ғжңүдёҖдәӣж–°зҡ„зү№зӮ№пјҢжҜ”еҰӮйңҖиҰҒViewHolderгҖӮжҲ‘д»¬йңҖиҰҒйҮҚеҶҷдёүдёӘдё»иҰҒж–№жі•пјҡз¬¬дёҖдёӘ(onCreateViewHolder)з”ЁжқҘжёІжҹ“и§Ҷеӣҫе’Ңе®ғд»¬зҡ„holderпјҢз¬¬дәҢдёӘ(onBindViewHolder)з”ЁжқҘдёәи§Ҷеӣҫз»‘е®ҡж•°жҚ®пјҢз¬¬дёүдёӘ(getItemCount)з”ЁжқҘиҝ”еӣһи§Ҷеӣҫж•°йҮҸгҖӮ

дёүгҖҒдҪҝз”ЁRecyclerView

дҪҝз”ЁRecyclerViewжңүд»ҘдёӢе…ій”®жӯҘйӘӨпјҡ

  1. е®ҡд№үз”ЁдҪңж•°жҚ®жәҗзҡ„жЁЎжқҝзұ»
  2. еңЁactivityдёӯж·»еҠ RecyclerViewжқҘеұ•зӨәitems
  3. еҲӣе»әдёҖдёӘиҮӘе®ҡд№үзҡ„зәҝжҖ§еёғеұҖXMLж–Үд»¶жқҘдҪҝitemеҸҜи§ҶеҢ–
  4. еҲӣе»әRecyclerView.Adapterе’ҢViewHolderжқҘжёІжҹ“item
  5. е°ҶйҖӮй…ҚеҷЁз»‘е®ҡеҲ°ж•°жҚ®жәҗжқҘеЎ«е……RecyclerView

дёӢйқўжҲ‘д»¬ж №жҚ®дёҠиҝ°жӯҘйӘӨиҜҰз»ҶеҲҶжһҗdemoдёӯзҡ„д»Јз ҒгҖӮ

в‘  еҲӣе»әжЁЎжқҝзұ»

жҜҸдёӘRecyclerViewйғҪз”ұж•°жҚ®жәҗж”ҜжҢҒгҖӮжҲ‘д»¬е®ҡд№үдёҖдёӘSayingпјҲжҲ‘зҢңиҝҷжҳҜвҖңиҜҙиҜҙвҖқзҡ„иӢұж–Үпјүзұ»дҪңдёәж•°жҚ®жЁЎжқҝпјҡ

в‘Ў еңЁеёғеұҖдёӯеҲӣе»әRecyclerView

еңЁres/layout/activity_main.xmlдёӯж·»еҠ RecyclerViewпјҢRecyclerViewе®һиҙЁдёҠиҝҳжҳҜдёҖдёӘз»„д»¶пјҢжүҖд»Ҙе…¶еңЁxmlж–Үд»¶дёӯзҡ„ж·»еҠ ж–№жі•е’ҢbuttonгҖҒtextViewзӯүз®ҖеҚ•з»„д»¶зӣёеҗҢгҖӮ

в‘ў еҲӣе»әиҮӘе®ҡд№үitemеёғеұҖ

еңЁеҲӣе»әadapterд№ӢеүҚпјҢжҲ‘д»¬йңҖиҰҒе®ҡд№үеҲ—иЎЁдёӯжҜҸдёҖиЎҢзҡ„еёғеұҖгҖӮжҲ‘еҶҷзҡ„жһҒдёәз®ҖпјҲchouпјүеҚ•пјҲlouпјүзҡ„еёғеұҖжҳҜдёҖдёӘз”ұеҸ‘йҖҒиҖ…зҡ„еҗҚеӯ—гҖҒеҸ‘йҖҒдҝЎжҒҜд»ҘеҸҠдёҖжқЎеҲҶеүІзәҝз»„жҲҗзҡ„ж°ҙе№іеёғеұҖгҖӮеңЁres/layout/item.xmlдёӯпјҡ

в‘Ј еҲӣе»әRecyclerView.Adapter

йҖӮй…ҚеҷЁзҡ„дҪңз”ЁжҳҜе°ҶжҹҗдёӘдҪҚзҪ®зҡ„еҜ№иұЎиҪ¬еҢ–дёәе°ҶиҰҒжҸ’е…ҘеҲ—иЎЁзҡ„иЎҢеҚ•е…ғпјҲThe adapterвҖҷs role is to convert an object at a position into a list row item to be inserted.пјү

з„¶иҖҢпјҢйҖӮй…ҚеҷЁиҝҳйңҖиҰҒдёҖдёӘViewHolderеҜ№иұЎжқҘвҖңжҸҸиҝ°жҜҸдёҖиЎҢдёӯжүҖжңүи§Ҷеӣҫе№¶жҸҗдҫӣжҜҸдёҖиЎҢдёӯжүҖжңүи§Ҷеӣҫзҡ„и®ҝй—®йҖ”еҫ„вҖқпјҲdescribes and provides access to all the views within each item row.пјүгҖӮ

жҲ‘д»¬ж–°е»әMyViewHolder.javaж–Үд»¶пјҡ

е®ҢжҲҗViewHolderеҗҺе°ұеҸҜд»ҘејҖе§ӢеҶҷAdapterдәҶпјҢж–°е»әMyAdapter.javaж–Үд»¶пјҡ

в‘Ө е°ҶAdapterз»‘е®ҡеҲ°RecyclerView

AdapterеҶҷе®ҢеҗҺе°ұеҸҜд»ҘеӣһеҲ°MainActivity.javaдёӯжңҖеҗҺе®Ңе–„жҲ‘д»¬зҡ„зЁӢеәҸдәҶгҖӮ

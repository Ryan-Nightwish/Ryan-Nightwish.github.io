Deep Residual Learning for Image Recognition

出处：CVPR2016

原文：Deep Residual Learning for Image Recognition

参考中文：ResNet论文翻译——中文版

摘要

1、深度神经网络训练起来十分困难。我们提供了一种残差学习框架，可以更轻松地训练比以往更深的网络结构。

2、不同于学习未参考（unreferenced）的函数，我们明确地将层重新定义为参考层的输入的学习残差函数。我们提供了全面的证据，表面这些残差网络更容易优化，并且可以随着网络深度明显增加，正确率也会更好。

3、在ImageNet数据集上，我们计算了一个152层的残差网络，这比VGG网络还要深8倍，但是复杂度更低。这些残差网络的集合在ImageNet测试集上取得了 3.57% 的误差。

介绍

1、深度卷积神经网络在图像分类领域带来了一系列的突破，它很自然地将不同层级的特征与分类器以端到端的形式进行集成，特征的“级别”可以由堆叠的层数（深度）来提升。

2、随着深度的增加，梯度消失和梯度爆炸往往会从一开始就阻碍收敛。然而，通过标准化初始化normalized initialization和中间标准化层的方法，这个问题很大程度上已经解决了。

ChatGPT：

首先，normalized initialization是一种权重初始化方法，它可以帮助避免梯度爆炸和梯度消失的问题。传统的权重方法中，通常会使用随机初始化来初始化权重。但是，这种方法可能会导致权重的值过大或过小，从而导致梯度爆炸或梯度消失的问题。为了解决这个问题，normalized initialization使用了一种特殊的初始化方法，它将权重的值限制在一个合适的范围内，从而避免梯度爆炸和梯度消失的问题。

其次，intermediate layers是一种在网络中添加归一化层的方法，它可以帮助解决梯度消失的问题。在深层网络中，由于梯度会不断地传递下去，可能会导致梯度的值变得非常小，从而导致梯度失的问题。为了解决这个问题，intermediate normalization layers在网络中添加了一些归一化层，例如Batch Normalization层，它可以将每一层的输入进行归一化，从而使得梯度的值保持在一个合适的范围内，避免梯度消失的问题。

3、但是，堆叠更多的层数是否就意味着可以学习到更好的网络呢？当深度网络开始收敛时，会暴露一个退化degradation问题：随着网络层数的增加，准确率会达到饱和，然后迅速下降。意外的是，这种退化并不是过拟合导致的。在一个适当的深度模型上添加更多的层会导致更高的训练误差。下图就是典型的例子。

4、退化表面不是所有的系统都可以类似地很容易被优化。想象有一个浅层架构和一个堆了更多层的深层架构，在构建深层架构的过程中有这么一种可能的方式：添加的层都是恒等映射（将输入直接传递到输出，而不进行任何计算），而其他的层是浅层架构的拷贝。这种可能的构造方法表面，更深的模型的训练误差不应该比其浅层模型更高。也就说，更深的模型应该至少能够达到与浅层模型相同的训练误差，如果不能达到，那么就需要虑其他的解决方案。可是实验表明，我们暂时无法找到比这个更好的解决方案。

也就是说深层网络训练出来一般来说都会比浅层网络有更多的损失、更大的训练误差。

5、我们通过引入深度残差学习框架来解决退化问题。我们明确地让这些层拟合残差映射，而不是希望每几个堆叠的层直接拟合期望的基础映射。形式上，将期望的基础映射表示为，我们将堆叠的非线性层拟合另一个映射。原始的映射重写为。作者在这里做了一个假设，即残差映射比原始的、未参考的映射更容易优化。比如在极端情况下，如果一个恒等映射是最优的，那么将残差置为零比通过一堆非线性层来拟合恒等映射更容易。具体的实现如下图所示，这样的好处有三点，一是不会增加新的参数，二是整个网络依然可以使用SGD进行训练，三是容易被通用的框架实现。

在增加了深度后，是理想的映射，作者认为这很难实现，并构造了一个新的近似映射，这样理想的映射就变成了。作者在这里先假设近似的映射更好学，并用了一个极端的情况进行证明。

6、通过在ImageNet和其他数据集上进行实验来解决退化问题并论证我们的方法，我们发现：

  • 我们极深的残差网络很容易优化，但简单堆叠的深度网络会有很高的训练误差。
  • 我们的深度残差网络可以很容易地从深度的增加中获得精度的提高，结果比以往的网络好很多。

残差学习

1、令是几个堆叠层需要拟合的基础映射，是这些层的输入。如果这些非线性层可以渐近近似一些复杂的函数，那么就等价于这些层可以渐近近似残差函数。所以我们明确让这些层去近似残差函数，而不是直接近似。那么原始函数就变成了。尽管两种形式都可以渐近近似想要的函数，但它们学习的难易程度不同。

实现

训练：

  • 调整图像大小，其较短边在之间随机采用，用于尺度增强
  • 从图像或其水平翻转中随机进行的裁剪，并逐像素减去均值
  • 进行标准颜色增强
  • 在卷积之后和激活函数之前使用批量归一化batch normalization
  • 初始化权重，训练简单、残差网络，使用批大小为256的SGD方法，训练次迭代
  • 学习率从0.1开始，误差稳定后除以10。权重衰减为0.0001，动量为0.9

ChatGPT：weight decay是什么？

在深度学习中，weight decay是一种正则化技术，它通过对模型的权重进行惩罚来避免过拟合。具体来说，weight decay会在模的损失函数中添加一个正则化项，这个正则化项是权重的平方和乘以超参数λ，即L2正则项。这个正则化项会使得模型的权重趋向于较小的值，从而避免过拟合。
